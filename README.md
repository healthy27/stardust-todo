# ✨ Stardust (별자리 투두)

> **"일정도 하나의 광활한 우주다"**
>
> 할 일을 별(Star)로, 성취를 별자리(Constellation)로 시각화한 감성 투두 리스트 앱입니다.

## 🌌 프로젝트 소개

**Stardust**는 삭막한 체크리스트 대신, 나의 노력이 밤하늘의 별이 되어 빛나는 경험을 제공합니다. 
단 1시간의 타임어택 해커톤으로 기획되고 개발된 MVP 프로젝트입니다.

### 핵심 기능

*   **나만의 우주 (Universe View)**: 할 일을 생성하면 우주에 별이 뜹니다. 드래그하여 자유롭게 배치할 수 있습니다.
*   **별자리 (Constellation)**: 할 일을 완료하면 별이 빛나며, 이전에 완료한 별과 선으로 연결되어 나만의 별자리가 됩니다.
*   **별똥별 보관소 (Meteor Archive)**: 제때 완료하지 못한 일들은 사라지지 않고 깊은 우주(별똥별 보관소)에 머뭅니다.
*   **감성 메시지**: 할 일을 생성하거나 완료할 때, 지친 하루를 위로하는 감성적인 메시지가 나타납니다.
